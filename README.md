# 🧹 Limpieza y Calidad de Datos para una Tienda de Ropa

## Descripción

Este proyecto tiene como objetivo identificar, analizar y corregir problemas de calidad de datos presentes en las bases de inventario y ventas de una tienda de ropa.

Los datasets contienen errores comunes encontrados en empresas reales, tales como:

- Registros duplicados
- Valores nulos
- Formatos inconsistentes
- Errores tipográficos
- Datos inválidos
- Inconsistencias entre tablas
- Outliers

El proceso fue desarrollado utilizando Python y Pandas.

---

## Objetivos

- Mejorar la calidad de los datos.
- Estandarizar formatos.
- Detectar registros duplicados.
- Corregir errores de escritura.
- Validar relaciones entre tablas.
- Generar un reporte de calidad de datos.

## Impacto de Negocio

La baja calidad de datos puede generar:

- Sobrestock o quiebres de inventario.
- Errores en reportes de ventas.
- Decisiones incorrectas de compra.
- Pérdida de confianza en los indicadores.

Este proyecto busca transformar datos inconsistentes en información confiable para la toma de decisiones.
